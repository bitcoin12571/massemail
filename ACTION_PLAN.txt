STEP 1: CREATE GMAIL APP PASSWORD (2 minutes)
═════════════════════════════════════════════════════════════════════════════

Go to: https://myaccount.google.com/

1. Click "Security" in the left menu
2. Enable "2-Step Verification" (if not already enabled)
3. Look for "App passwords" section
4. Select "Mail" as the app and your device type
5. Click "Generate"
6. Google will show you a 16-character password
7. ⭐ COPY THIS PASSWORD


STEP 2: CONFIGURE VERCEL (2 minutes)
═════════════════════════════════════════════════════════════════════════════

Go to: https://vercel.com/dashboard

1. Find and click on "email-dashboard" project
2. Click "Settings" tab at the top
3. Click "Environment Variables" in the left sidebar
4. Add these 5 NEW environment variables:

   Variable Name:        EMAIL_PROVIDER
   Value:                gmail
   [Save]

   Variable Name:        SENDER_NAME
   Value:                Your Company Name
   [Save]

   Variable Name:        EMAIL_FROM
   Value:                your-email@gmail.com
   [Save]

   Variable Name:        SMTP_USER
   Value:                your-email@gmail.com
   [Save]

   Variable Name:        SMTP_PASS
   Value:                (paste the 16-character password from Step 1)
   [Save]

5. Wait for "Environment Variables Updated" confirmation


STEP 3: REDEPLOY YOUR APPLICATION (1 minute)
═════════════════════════════════════════════════════════════════════════════

Choose ONE of these options:

OPTION A (Easiest - Use Dashboard):
  1. Stay in Vercel Dashboard
  2. Click on "Deployments" tab
  3. Find the most recent deployment
  4. Click the "..." menu
  5. Click "Redeploy"
  6. Wait for deployment to complete

OPTION B (Use CLI):
  1. Open terminal
  2. Run: vercel deploy --prod
  3. Wait for deployment to complete

OPTION C (Use Script):
  1. Open terminal in your project
  2. Run: ./deploy.sh
  3. Follow the prompts


STEP 4: TEST YOUR EMAIL CONFIGURATION (1 minute)
═════════════════════════════════════════════════════════════════════════════

1. Open your Vercel deployment URL
2. Navigate to Settings or Email Configuration section
3. Click "Test Connection" button
4. You should see: ✅ "Gmail connection verified successfully"
5. Create a test contact or use existing one
6. Send a test email
7. Check your inbox for the test email


✅ DONE! Your emails will now work!

TROUBLESHOOTING

If Step 4 shows an error:

ERROR: "Email delivery is not configured"
  ✓ Verify all 5 environment variables are set in Vercel
  ✓ Wait 30 seconds after saving variables
  ✓ Redeploy the application
  ✓ Refresh the page

ERROR: "Could not send email"
  ✓ Check that SMTP_PASS is exactly the 16-character App Password
  ✓ Verify SMTP_USER matches EMAIL_FROM
  ✓ Check that EMAIL_PROVIDER = "gmail" (case sensitive)
  ✓ Look at Vercel logs for "[EMAIL SERVICE]" error messages

EMAIL STILL NOT RECEIVED:
  ✓ Check spam/junk folder
  ✓ Verify the recipient email address is correct
  ✓ Check Vercel function logs for detailed errors
  ✓ Read EMAIL_CONFIG_FIX.md for more troubleshooting
